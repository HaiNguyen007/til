# Primitives

```
string
number
null // typeof null === 'object'
symbol
undefined
```

- **Wrapper** to **Primitive**: `new String('hello')`
- **Primitive** to **Wrapper** (`Object.prototype.valueOf`): `new String('hello').valueOf()`
